# Content Management CRUD Application

A RESTful CRUD web application built using **Node.js** and **Express.js**, with a simple frontend developed using **HTML, CSS, and JavaScript**. The application allows users to create a unique ID and manage content through create, read, update, and delete operations.

---

## 🚀 Features

- Create a user ID and add content
- Read stored content
- Update existing content
- Delete content
- REST API based backend
- Simple and clean frontend UI
- Runs locally on localhost

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Frontend:** HTML, CSS, JavaScript  
- **API Style:** RESTful APIs  
- **Version Control:** Git & GitHub
